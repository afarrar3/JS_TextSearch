GIS Address Lookup

Enter an address in the textbox and click "Find" to view which jurisdiction an address lies within. If no address is found, then the address, as entered, does not exist within our database. This will verify if an address lies within the Frederick County, Stephens City, or Middletown boundaries. Alternatively, type in a street name to return all addresses for that given street.



Address Format: House #, Street Name, Abbrv. Suffix (e.g. 1080 Coverstone Dr)
